# 个人博客

## 基本文章的格式（front matter）
```md
---
title: 文章标题
date: '2019-08-08 08:00:00' // 写作时间
sidebar: 'auto'             // 自动配置侧边栏
categories:                 // 文章分类
 - typescript
 - javascript
tags:                       // 文章标签
 - react
 - vue
 - node
keys:                       // 配置文章密码
 - '123456'
publish: false              
---
{content}
```
